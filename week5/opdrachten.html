Opdrachten week 5

Inleiding

Op deze pagina vind je de opdrachten voor week 5.

Elke week zijn er een aantal makkelijke, medium en moeilijke opdrachten waar je in het practicum aan gaat werken. Aan het begin kan je de uitwerking van de opdrachten aan je docent laten zien. Deze zal ze afvinken en je zult hiervoor avans punten krijgen.

Voor het huiswerk van elke weeg staat er een start project klaar. Deze startprojecten zijn onderdeel van de github repository waarmee dit webdictaat gemaakt is. Om deze start projecten te gebruiken kun je het beste de github repo clonen. Rechts van dit informatie blok staat een link en een clone button. Voor elk project is meteen een uitwerking toegevoegd. Gebruik deze om te spieken als je er even niet meer uit komt.


Daarnaast bevat elke opdracht, ook een Unit test project en een aantal interface objecten. Gebruik deze interfaces om je eigen objecten te immplementeren, en run de tests om te controlleren of de opdracht goed is.

De workshop opdrachten zijn tevens huiswerk opdrachten. Deze opdrachten zijn oefenmateriaal voor het assessment aan het einde van dit blok. De opdrachten zijn niet verplicht, maar de eindopdracht wordt pittig. Ik kan je aanraden deze opdrachten dus netjes bij te houden.

De opdrachten van deze week geven je een kijkje in de keuken van Entity Framework. De 3 manieren voor het werken met Entity Framework worden toegepast.

  • Model First
  • Code First
  • Database First
